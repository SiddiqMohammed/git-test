GIT Cheat sheet

Create file 
  touch <filename.extenstion>
 
Add files to git 
  git add .

Commit files
  git commit -m 'description'

push files
  git push

Create branch
  git branch <branchname>

open branch
  git checkout <nameofbranch>

Merge branch into origin 
-> goto master branch
  git merge <branchname>
  
check status of files in folder
  git status
